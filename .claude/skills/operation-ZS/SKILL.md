# 招商业务处理工具集

Excel 数据处理工具的集合，通过命令行参数调用。

## 快速开始

### 安装依赖

```bash
pip install pandas openpyxl python-dateutil
```

### 使用流程

**执行步骤：**
1. 调用 skill
2. **使用 AskUserQuestion 收集参数**（工具选择、月份、文件路径）
3. 执行处理

## 交互协议（强制）

**重要**：每次技能调用必须使用 `AskUserQuestion` 交互收集参数：

1. **工具选择**：选择要使用的工具
2. **年月参数**：YYYYMM 格式（如 202601）
3. **文件路径**：逐一收集 4 个文件路径（原始数据、历史数据、医院表、底价表）

**禁止**：
- ❌ 使用纯文本输出让用户"告诉我路径"
- ❌ 自动检查桌面文件是否存在
- ❌ 使用默认路径直接执行

### 命令行调用

```bash
python3 ${SKILL_DIR}/scripts/ts_data_clean.py -m 202601 \
  -r 原始数据.xlsx \
  -c 历史数据.xlsx \
  -H 医院表.xlsx \
  -p 底价表.xlsx
```

## ts_data_clean - TS数据清洗

### 处理流程

1. 读取原始招商数据文件
2. 筛选指定年月的数据
3. 追加到已清洗的历史数据中
4. 匹配医院信息表（CRM医院编码）
5. 匹配底价表（26年底价）
6. 计算实际底价和底价额
7. 应用特殊产品底价调整
8. 导出清洗后的数据

### 必须参数

| 参数 | 简写 | 说明 |
|------|------|------|
| `--month` | `-m` | 年月 (YYYYMM)，**必须指定** |
| `--raw-data` | `-r` | 原始招商数据路径，**必须提供** |
| `--clean-data` | `-c` | 已清洗历史数据路径，**必须提供** |
| `--hospital-table` | `-H` | 医院信息表路径，**必须提供** |
| `--price-table` | `-p` | 底价表路径，**必须提供** |

### 可选参数

| 参数 | 简写 | 说明 |
|------|------|------|
| `--output` | `-o` | 输出文件名（不含扩展名） |
| `--output-dir` | `-d` | 输出目录（默认：输入文件所在目录） |

### 特殊定价规则

| 产品 | 区域 | 单价条件 | 26年实际底价 |
|------|------|----------|-------------|
| 吡美莫司14g | - | 单价 = 39.34 或 42 | 保持原单价 |
| 吡美莫司14g | 上海 | 单价 ≠ 39.34/42 | 38 |
| 他克莫司10g | 江西 | - | 15.6 |
| 他克莫司10g/0.03% | 江西 | - | 9.48 |

## price_sales_progress - 底价销售进度

### 处理流程

1. 读取主表（26年招商销售进度公示.xlsx）
2. 读取指标表（T2026-招商-一维表.xlsx），筛选医疗端数据，计算26年累计指标
3. 读取流向数据（TS{月份}-招商（已清洗）.xlsx）
4. 计算26年累计销售额（医疗端、保留数据口径）
5. 计算25年同期累计销售额（医疗端、保留数据口径）
6. 计算达成率、销售进度、同比增长额
7. 添加总计行（使用原始数据计算后再四舍五入）并导出

### 必须参数

| 参数 | 简写 | 说明 |
|------|------|------|
| `--month` | `-m` | 当前年月 (YYYYMM)，**必须指定** |
| `--template` | `-t` | 主表路径（26年招商销售进度公示.xlsx），**必须提供** |
| `--flow-data` | `-f` | 流向数据路径（TS{月份}-招商（已清洗）.xlsx），**必须提供** |
| `--indicator-table` | `-i` | 指标表路径（T2026-招商-一维表.xlsx），**必须提供** |

### 可选参数

| 参数 | 简写 | 说明 |
|------|------|------|
| `--output` | `-o` | 输出文件名（不含扩展名） |
| `--output-dir` | `-d` | 输出目录（默认：流向数据所在目录） |

### 计算字段说明

| 字段 | 说明 | 格式 |
|------|------|------|
| 26年部门考核指标 | 主表中的年度考核指标 | 数值，1位小数 |
| 26年累计指标 | 指标表中202601~当前月，**医疗端**（渠道类型='医疗'），按省区汇总底价指标 | 数值，1位小数 |
| 25年累计销售额 | 流向数据中去年同期，**医疗端+保留口径**，按省区汇总26年底价额(万) | 数值，1位小数 |
| 26年累计销售额 | 流向数据中202601~当前月，**医疗端+保留口径**，按省区汇总26年底价额(万) | 数值，1位小数 |
| 26年累计达成率 | 26年累计销售额 / 26年累计指标 × 100 | 百分比，2位小数 |
| 26年销售进度 | 26年累计销售额 / 26年部门考核指标 × 100 | 百分比，2位小数 |
| 26年累计同比增长额 | 26年累计销售额 - 25年累计销售额 | 数值，1位小数 |

### 输出列顺序

```
省区 | 招商经理 | 26年部门考核指标 | 26年累计指标 | 25年累计销售额 | 26年累计销售额 | 26年累计达成率 | 26年销售进度 | 26年累计同比增长额
```

### 命令行示例

```bash

```
