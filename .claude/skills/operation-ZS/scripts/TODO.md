## 功能1：根据规则标记金纽尔零售剔除+火把花全部剔除

### 需求说明
- **金纽尔零售剔除**：根据特定规则标记金纽尔产品在零售渠道的数据需要剔除
- **火把花全部剔除**：火把花产品的所有数据需要剔除

### 涉及字段
- 产品名称（商名S/产品）
- 终端类型（皮肤-终端类型以此为准！）
- 数据口径标记字段

### 实现思路
```python
# 伪代码示例
# 火把花全部剔除
is_huobahua = df['商名S'].str.contains('火把花', na=False)
df.loc[is_huobahua, '数据口径'] = '剔除'

# 金纽尔零售剔除
is_jinniuer = df['商名S'].str.contains('金纽尔', na=False)
is_retail = df['皮肤-终端类型以此为准！'].isin(['零售', '非医疗'])
df.loc[is_jinniuer & is_retail, '数据口径'] = '剔除'
```

## 功能2：区分商业类型，主流/现款/其他等

### 需求说明
根据代理商信息区分商业类型：
- **主流**：主流商业公司
- **现款**：现款交易
- **其他**：其他类型

### 涉及字段
- 代理商(剔除票折)
- 商业类型（新增字段）

### 实现思路
```python
# 伪代码示例
# 定义商业类型映射规则
commercial_type_map = {
    # 主流商业
    '国药': '主流',
    '华润': '主流',
    '上海医药': '主流',
    # 现款
    '现款': '现款',
    # 其他...
}

# 根据代理商名称匹配商业类型
def get_commercial_type(agent):
    for keyword, type_ in commercial_type_map.items():
        if keyword in agent:
            return type_
    return '其他'

df['商业类型'] = df['代理商(剔除票折)'].apply(get_commercial_type)
```
